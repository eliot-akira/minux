
Welcome to Minux, a cozy operating system on a RISC-V virtual machine.

Networking supports HTTP/HTTPS but only endpoints that allow cross-origin requests. There is no data persistence currently, or import/export.

CTRL/CMD + SHIFT + V to paste text

Some commands for example:

htop                    ncdu /
df                      dmesg
bc                      tmux
cmatrix                 sqlite3
